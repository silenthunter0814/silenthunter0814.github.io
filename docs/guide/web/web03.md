## 0 ES6 速查表

1. 模板文字将表达式嵌入到字符串中：  
`${ninja}`

2. 块作用域变量：  
  - 使用新的let关键字创建块级变量：  
  `let ninja = “Yoshi”`
  - 使用 `const` 关键字创建块级常量变量，其值不能重新分配为全新值：
  `const ninja = "Yoshi"`
